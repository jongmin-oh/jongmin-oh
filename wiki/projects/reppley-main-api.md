# 레플리 메인 — 캐릭터 페르소나 챗봇 서비스 API

**기간**: 2024.01 – 2026.05  
**소속**: [[entities/reppley\|Reppley]]  
**역할**: 설계·구현·운영 전담

## 개요

레플리 메인 제품. 캐릭터 페르소나 챗봇 서비스의 핵심 API. 시장에서 검증된 기능 패턴을 빠르게 흡수해 구현하고, 자사 사용자로 재검증하는 사이클로 운영.

## 주요 기능 5종

| 기능 | 설명 |
|------|------|
| 캐릭터 챗 | 상황·페르소나에 맞는 캐릭터 응답 생성 및 스토리 진행 |
| 속마음 엿보기 | 현재 상황에 맞는 캐릭터의 속마음 생성 |
| 사진 보내기 | 사용자가 보낸 사진을 캐릭터가 이해하고 답변 |
| 페르소나 만들기/다듬기 | 키워드만으로 캐릭터 페르소나 자동 생성 |
| 상황 이미지 | 대화 컨텍스트에 맞는 캐릭터 이미지 노출 |

## 핵심 결정들

### 비용 최적화
일 70만 건 규모에서 비용이 선형 증가하지 않도록:
- **대화 요약 시스템** 도입 — 컨텍스트 축약으로 토큰 절감
- **프롬프트 캐싱** 도입
- 합산 **건당 비용 30% 절감**

### 시장 패턴 흡수
경쟁 제품(캐릭터 챗봇 시장) UX·기능을 분석 → 자사 제품에 맞게 빠르게 구현 → 자사 사용자에게 검증하는 사이클.

## 스택

`AWS ECS` · `AWS ElastiCache` · `Python` · `FastAPI`

- TIPS · 예비창업패키지 · 구글 창구 선정
