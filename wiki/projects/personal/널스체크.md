# 널스체크 (Nurse Burnout Check)

**성격**: 1인 개발 · 신규·저연차 간호사를 위한 소진·이직·회복 의사결정 지원 서비스  
**배포**: https://번아웃.com/

## 왜 만들었나

> 1년차 간호사인 동생이 야간 교대 후 "나 지금 버텨도 되는 상태야, 아니면 뭔가 잘못되고 있는 거야?"라고 물었을 때 제대로 답하지 못했다.

일반 번아웃 테스트만 있을 뿐, 교대근무 피로·프리셉팅 충격·환자안전 부담·경제적 불안 같은 **간호사 특유의 맥락**을 다루는 도구가 없어 직접 제작.

## 문제 정의

- "힘들다"는 감각은 있지만 **버텨도 되는 힘듦**인지 **위험 신호**인지 구분 어려움
- 퇴사하고 싶은 마음 vs 경제적 불안 동시 존재
- "간호 자체가 안 맞는지" vs "지금 병원/부서가 안 맞는지" 분리 불가

## 해결책

### 간호사 특화 5차원 자가점검 (25문항, 약 4분)

| 차원 | 내용 |
|------|------|
| 개인 번아웃 | |
| 업무 번아웃 | |
| 정서 위험 | |
| 이직 압력 | |
| 환경 불일치 | |

일반 테스트에 없는 문항: 교대근무 후 회복 불가, 프리셉터 관계, 실수 공포, 출근 전 신체 반응 등.

### 점수가 아닌 결정 지원

결과를 **유형으로 분류** → 유형별 다음 행동 추천:
- 고위험 소진형
- 환경 불일치형
- 준비된 이직 필요형
- 직무 재설계형

고위험 응답 시 위기 상담 자원 즉시 안내.

### 결과 공유 링크

결과 페이지를 링크로 공유 → 동료·가족과 함께 확인 가능.

## 스택

`React 19` · `TypeScript` · `Vite` · `Plain CSS (design tokens)` · `Playwright (E2E)` · `S3 + CloudFront`  
디자인: Google Stitch

## 의의

- "단순 점수 측정"이 아니라 **"현재 상태 이해 + 다음 행동"** 연결 → 감정이 아닌 데이터 기반 의사결정 도구
- 가까운 사람의 실제 고민에서 출발해 문제를 정의하고 직접 서비스로 만들어 배포
- 인프라: S3 + CloudFront 서버리스 호스팅 → [[skills/working-style\|서버리스 우선]] 철학 일관 적용
